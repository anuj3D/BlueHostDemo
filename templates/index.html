About This Demo ▼

This interactive demo showcases the capability to dynamically generate and customize a product display website using simple data files. Key features include:

  • Dynamic Content: All products (titles, descriptions, images, prices) are loaded from a CSV, making the content easily updatable.
  • Visual Customization: Switch between distinct "Amazon Style" and "eBay Style" layouts with a single click.
  • User Personalization: Products reorder based on hardcoded user preferences (e.g., "Tech Enthusiast" vs. "Home Maker"), demonstrating tailored experiences.
  • Product Details Pages: Click any product to view full details and discover similar items.
  • Logging: Tracks user interactions like product clicks and "Add to Cart" actions for analytics.

Technology Stack:

  • Backend: Python with Flask (handles CSV processing, data serving, logging).
  • Frontend: HTML, CSS, JavaScript (for dynamic rendering, style switching, and interactivity).
  • Templating: Jinja2 (used by Flask to render HTML).
  • Deployment: Hosted on Render's free tier, showcasing easy cloud deployment.

This demo can be extended to support real-time inventory updates, A/B testing, various user segments, and more complex data-driven web applications.
